p

Paragraph element. Can house any group of content, but usually blocks of text.

<p>Example</p>

Back To Top

ul

Unordered list. Creates a bulleted list of unordered items.

<ul> <li>Example</li> </ul>

Back To Top

ol

Ordered list. Creates a numbered list of ordered items.

<ol> <li>Example</li> </ol>

Back To Top

li

A list item in either an ordered or unordered list, being an attribute of both.

<li>Example</li>

Back To Top

a

Creates a hyperlink to another webpage, a different section of the same webpage, an email address, etc. A necessary attribute is "href," where the link is housed.

<a href="Example"></a>

Back To Top

img

Places an embedded image in the document. Attributes include "src," which houses the link, and "alt," which provides a text alternative to the image.

<img src="Example" alt="Example"/>

Back To Top

html

HTML is the root element of a document from which all other elements must descend.

<html>Example</html>

Back To Top

head

The head element houses metadata such as the document's title and links to stylesheets.

<head>Example</head>

Back To Top

link

The link element is used in the head section to connect the document with accompanying stylesheets.

<head> <link rel="stylesheet" href="Example"/> </head>

Back To Top

title

The title element is used under the head section as metadata for the document's title in browsers and search engines.

<head> <title>Example</title> </head>

Back To Top

body

The body element denotes where the document's visible content begins.

<body>Example</body>

Back To Top

span

The span element is a container used to group elements together under "class" or "id."

<span class="Example">Example</span>

Back To Top

div

The div element serves a similar purpose to "span," grouping elements together for later styling in CSS.

<div class="Example">Example</div>

Back To Top

header

Elements under the header tag represent the document's literal header, typically including navigation, search bar, etc.

<header>Example</header>

Back To Top

nav

Hosts the navigational elements of the document, usually under header.

<nav class="Example"> <ol> <li class="Example"><a href="Example">Bikes</a></li> </ol> </nav>

Back To Top

footer

Elements under the footer tag represent the document's literal footer, typically including info about the author and copyright.

<footer>Example</footer>

Back To Top

main

The main tag is used to section off the most important content of the document.

<main role="main">Example</main>

Back To Top

section

The section tag is used to represent any standalone section of the document which cannot be represented otherwise.

<section>Example</section>

Back To Top

aside

An aside tag represents content indirectly related to the document, usually as sidebars.

<aside>Example</aside>
